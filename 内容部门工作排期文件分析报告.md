### 1.2 功效与产出当量

功效以**任务完成当量**衡量，按交付状态赋权：

| 交付状态 | 完成当量权重 |
| --- | --- |
| 完成 | 1.0 |
| 执行中 | 0.5 |
| 延期 | 0.3 |
| 搁置 | 0.0 |
| 空白/未标注 | 0.5（按进行中处理） |

### 1.3 平均劳动生产率

部门平均劳动生产率 = 全部门任务完成当量 ÷ 全部门总工时，单位为 **任务当量/小时**。

人均劳动生产率 = 个人任务完成当量 ÷ 个人总工时，单位为 **任务当量/小时**。

## 五、关键发现与建议

1. **工时分布不均**：部门总工时约 1961 小时，人均约 123 小时。其中 吕利尧 负荷最高（332 小时，占比 16.9%），林子微 负荷最低（33 小时）。建议复核任务分配，避免长期高负荷人员出现瓶颈。
2. **项目形式集中度高**：工时前三类为 视频/视频播客、播客、商务，合计占比达 65.0%。资源投入与部门核心产出方向基本一致。
3. **整体交付率**：当前完成率为 60.1%，尚有大量任务处于“执行中/延期/搁置”状态。建议对延期与搁置任务逐条复盘，明确阻塞点。
4. **人效标杆**：沈放 的劳动生产率最高（0.535 当量/小时），可作为同类任务的标准工时与流程参考。
5. **低完成率/低生产率人员**：乔培远 生产率最低（0.119 当量/小时），需排查其任务难度、协作阻塞或工时估算偏差。
6. **交付质量突出**：赵雁玲 的交付完成率最高（86.4%），可总结其项目推进经验并在部门内分享。
