## 段1：各卷主题与情绪曲线
全篇共5章，每章可看作一个独立的“卷”，情绪曲线逐章递进。第1章《骸骨》——冷开局，林远在开会被赵鹏宣布剽窃，愤怒与压抑交织，章末发现时间戳矛盾，情绪由堵转喘。第2章《暗礁》——低气压，林远试图说服前同事作证，反被赵鹏设局陷害，证据链断裂，情绪沉入谷底；章末通过监控录像发现宋副总监的影子，埋下转折。第3章《裂痕》——张力陡增，林远从财务离职员工处拿到一条银行流水线索，却面临被公司开除的危机；同时赵鹏主动约谈，试图收买林远，在一次谈判中两人情绪对撞，赵鹏首次泄露自己的动机（母亲重病），角色灰色地带浮现，情绪微暖但危机更深。第4章《博弈》——高烈度对抗，林远设下“假文件”陷阱引诱赵鹏上钩，同时暗中向盛达集团法务发匿名线索；章末赵鹏发现被骗，反手将林远与宋副总监的通话录音池公开，林远陷入全面被动，情绪压强达顶峰。第5章《清算》——冷冽收束，林远利用之前藏好的备份证据（财务报表扫描件）在投资人路演前夜直接报警，同时联合盛达集团内线完成致命一击；赵鹏被抓后供出宋副总监，但林远也因私自录音被公司起诉，最后和解离开。情绪曲线：压抑→爆裂→喘息→高压→冷寂。

## 段2：卷间钩子与回收承诺（前台/后台双层）
前台钩子：第1章埋下“时间戳矛盾”钩子，在第2章通过前同事删除记录证实剽窃但失败，第4章林远用同一线索反制赵鹏（伪造时间戳文件诱捕）。第2章出现“赵鹏曾去财务部深夜加班”的监控片段，后台钩子在此初现——指向宋副总监指使操作对公账户，第5章回收为赵鹏承认被指使的关键证据。后台主线钩子：第1章末尾林远无意中发现赵鹏U盘里有“盛达集团融资计划”的修订版，文件名命名规则异常（A3-17-06），第3章揭示这是宋副总监资产转移项目的代号，第5章成为境外账户追查的起点。第2章宋副总监通过HR向林远施压（“私查同事档案”警告），后台钩子暗示公司内控已被控制，第4章回收为林远利用HR的恐惧心理反制宋副总监（拿到宋副总监与法务的违规报销记录）。核心后台钩子仅1条（腐败链条），其余钩子均为前台中短期，无平行宇宙。

## 段3：各卷OKR（Objective + Key Results）
全书Objective：林远达成“揭露赵鹏剽窃与宋副总监经济犯罪，获得盛达集团解约追责判决书，并与至少一个核心盟友建立长期合作”。

第1章OKR：
- O：确认剽窃事实并建立初步证据链。
- KR1：拿到TS核心创意的原始时间戳（通过云笔记历史版本）。
- KR2：确认赵鹏在开会上台展示的PPT内容与林远版本完全一致。
- KR3：发现赵鹏U盘中隐藏的“盛达集团融资计划”修订版文件名异常（后台钩子种子）。
- 附注：林远判断失误一次（以为前同事会直接作证，对方拒绝）。

第2章OKR：
- O：在赵鹏反击中稳住局面，并查出一条指向宋副总监的线索。
- KR1：挖出赵鹏深夜进出财务部的监控片段（带来后台嫌疑）。
- KR2：林远被赵鹏伪造聊天记录陷害，在公司内部声誉受损，但通过监控自证清白。
- KR3：联系上已离职的财务专员小吴，获得一条模糊的境外账户流水截图（KR3依赖第1章KR3）。
- 附注：赵鹏动机初显（母亲重病，由对话片段暗示）。

第3章OKR：
- O：迫使赵鹏主动接触，暴露背后指使者，并避免被公司开除。
- KR1：林远约谈赵鹏，在录音中迫使赵鹏承认“是有人让我这么做的”（但不提具体名字）。
- KR2：宋副总监通过HR试图解雇林远（理由“私查同事档案”），林远用第2章获得的流水截图反面威胁HR停止动作。
- KR3：林远正式联系盛达集团内部法务主管张咏，匿名提供“融资计划异常”线索（后台链条初步外泄）。

第4章OKR：
- O：在赵鹏全面反击前设下陷阱，同时将后台线索传递给关键外部力量。
- KR1：林远伪造一份“宋副总监受贿备忘录”，诱使赵鹏偷取并作为证据与宋副总监谈判。
- KR2：赵鹏发现被骗后，公开播放林远与宋副总监的旧通话录音（被剪辑），林远陷入信用危机。
- KR3：盛达集团法务张咏暗中回复林远，表示已启动内部调查（前台博弈与后台线索汇合）。

第5章OKR：
- O：在投资人路演前完成最终揭露，摧毁腐败链条并脱身。
- KR1：林远在路演前夜报警，提交之前藏好的备份证据（包含第2章流水截图+第3章录音+第1章原始时间戳）。
- KR2：赵鹏被捕后供出宋副总监，腐败链条被公开。
- KR3：盛达集团撤销对赵鹏项目的融资，并公开追责林远所在公司的违规行为；林远拿到解约判决书，并与张咏签约新公司合作协议。
- 附注：林远因私自录音被公司起诉民事赔偿，最终和解（付出代价）。

## 段4：卷尾必须发生的改变
第1章末：林远从“被剽窃的受害者”转变为“主动调查者”，公司内部对他的态度从同情转为怀疑（赵鹏开始散布谣言）。第2章末：林远失去前同事这一潜在盟友，但获得外部线索（小吴的流水截图），公司正式对他下发警告函。第3章末：赵鹏首次暴露软肋（母亲病重），林远与宋副总监的关系从“看不见的幕后”变为“半公开对抗”，HR停止开除动作但将林远边缘化。第4章末：林远与赵鹏的关系破裂到不可修复，通话录音公开导致林远在行业内信用受损，但盛达集团内部调查启动，宋副总监开始感到压力。第5章末：赵鹏被捕，宋副总监被立案，林远离开原公司但以新身份开始创业，前女友与他彻底决裂。

## 段5：节奏原则
1. 高潮间距：全书共5章，大高潮（第4章末、第5章初）之间仅隔2章，中高潮每章章末至少有一个反转（信息暴露/对手反击/关系断裂）。
2. 喘息频率：高压段（第2-4章连续）每章必须包含一个约400字的喘息窗口（如林远在便利店吃盒饭时的内心独白，或与前女友短信对话），用于情绪缓冲和线索回顾。
3. 钩子密度：每章章末留1个主钩（必在最后300字内），主钩最长可悬2章（第1章时间戳钩子留到第3章才有实质进展），次钩（如角色关系）可悬到终章。
4. 信息释放节奏：前1/3（第1-2章）释放40%后台线索（剽窃事实+宋副总监影子），中段（第3-4章）释放50%（腐败链条细节+赵鹏动机），后1/3（第5章）释放最后10%（赵鹏供词+结局定论）。
5. 爽点节奏：每章至少1次林远与对手的直接言语或行动交锋（第1章当面质问，第2章举报反击，第3章谈判博弈，第4章陷阱对决，第5章报警收网），且每次交锋都要让局势短期扭转（不是单方面碾压）。
6. 情感节点递进：林远与前女友（赵鹏妹妹）的关系每章应有1次实质变化（第1章她试图调和→第2章她帮赵鹏质问林远→第3章她同情林远但选择沉默→第4章她向赵鹏通风报信→第5章彻底离开）；林远与盟友张咏的关系在第3章建立信任，第4章得到初步信任，第5章达成合作，每章必须有一次可信的互动推进。
