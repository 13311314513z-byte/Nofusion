## 核心标签
“灰人”、“被裹挟的野心家”、“消耗品”——他不是天生的恶棍，而是一个在生存压力下选择妥协的普通人。

## 反差细节
在办公室穿着廉价西装、戴假表，却在私下收藏正版乐高；剽窃时眼神毫不退缩，但在深夜接到母亲病危电话时会崩溃大哭。

## 人物小传（过往经历）
重庆本地人，单亲家庭，母亲做清洁工供他上大学。毕业后进广告公司，被林远手把手带出来。三年前母亲查出癌症，他为了凑手术费私下接私单被宋副总监抓住把柄，从此被控制。剽窃TS是宋副总监对他下达的“投名状”——成功后他不仅能还清债务，还能升任创意总监。

## 当前现状（第0章初始状态）
29岁，已婚无孩，妻子是老家护士。母亲在重症监护室，每月医药费1万6。他已经在宋副总监的授意下替换了林远的TS文件，内心矛盾但不敢反抗。

## 关系网络
- 林远：师父兼对手，内心深处仍尊敬林远，但利益冲突让他选择剽窃。
- 宋副总监：操控者和老板，赵鹏既怕他又想利用他获得升职。
- 赵曼（妹妹）：林远的前女友，赵鹏利用妹妹试图影响林远，但也有保护妹妹的本能。
- 财务小吴：曾帮他做过一次假账，后来小吴离职，赵鹏一直心虚。

## 内在驱动
想要摆脱病痛和债务，获得经济自由和职业地位。愿意为此付出尊严和良心，但最终发现换来的只是另一副镣铐。

## 成长弧光
从被动受控到主动尝试反噬宋副总监（第4章偷录音假文件），但最终失败被捕。他在审讯室第一次向林远说了“对不起”，完成了从“工具人”到“有自我意识但有罪的普通人”的转变。
