# 0609 Nofusion 测试计划 完善建议

> 基于对 `0609Nofusion测试计划.md` 的逐模块 review，结合项目代码现状（v1.4.1 + 0608 全部改动）和已知问题，提出 12 项结构性完善建议。

---

## 一、当前测试计划的质量评估

### 1.1 优点

| 维度 | 评价 |
|------|------|
| **覆盖度** | 9 个模块、~90 个用例，正向流程覆盖较全 |
| **结构** | 表格清晰，操作-预期-实际三列便于执行 |
| **闭环** | 缺陷记录 + 测试指标 + 环境记录，可追溯 |
| **场景化** | 模块 9 的综合场景设计合理，验证端到端 |

### 1.2 主要缺陷

| 缺陷类型 | 数量 | 严重程度 |
|---------|------|---------|
| **测试未实现功能**（5.1.4–5.1.6） | 3 项 | 🔴 高 |
| **缺失模块**（Genre/Truth/Runtime/Fanfic/Radar 等） | 5+ 个 section | 🟠 中 |
| **异常/负面测试缺失** | ~15 项应补充 | 🟠 中 |
| **边界条件覆盖不足** | 大文本/空状态/超长输入 | 🟡 低 |

## 二、12 项完善建议

### 建议 1：删除/标记尚未实现的功能测试项（🔴 紧急）

**问题**：模块 5.1.4–5.1.6 测试的是 `模型通用写作参数提升执行计划.md` 中规划的 Phase 1-4 功能，当前代码中**不存在**这些 UI。

```
5.1.4 高级参数 - top_p   ❌ 尚未实现
5.1.5 高级参数 - seed    ❌ 尚未实现
5.1.6 预设应用           ❌ 尚未实现
```

**处理方式**：
- 方案 A（推荐）：移到「附录：待实现功能验证清单」，标注"依赖 `模型通用写作参数提升执行计划.md` 完成后方可执行"
- 方案 B：保留但将预期结果改为「当前版本无此 UI，应显示 temperature 滑块」，验证"功能未暴露"而非"功能可用"

### 建议 2：补充缺失的 BookWorkspace Section 测试（🟠 重要）

当前模块 6 仅覆盖了角色/伏笔/场景/目标/导入，但 BookWorkspace 实际有 **14 个 section**，以下缺失：

| 缺失 Section | 建议加入模块 | 关键测试点 |
|-------------|-------------|-----------|
| **Overview（概览）** | 模块 6 | 书籍元数据显示、进度统计、最近编辑 |
| **Truth（世界观/真相）** | 模块 6 | TruthFiles 集成、条目 CRUD |
| **Summaries（摘要）** | 模块 6 | 世界观/主角/群像摘要渲染 |
| **Runtime（运行时状态）** | 模块 6 | MemoryDB 查看、全局状态刷新 |
| **Fanfic（同人设定）** | 模块 6 | Canon 设定导入、冲突检测 |
| **Sources（来源材料）** | 模块 6 | 来源文件列表、删除 |
| **Chat（书籍内聊天）** | 模块 2 | BookSidebar 联动、会话切换 |

> 注：部分 section 当前为轻量包装（如 `BookTruthSection.tsx` 仅 25 行），但仍需验证集成是否断裂。

### 建议 4：补充异常/负面测试用例（🟠 重要）

当前计划 90%+ 是正向流程。建议每个核心模块补充 2-3 个异常用例：

**模块 1（新建书籍）**：
| 1.6 | 超长标题 | 输入 200 字标题 | 截断或报错 |
| 1.7 | 特殊字符标题 | 输入 `<script>alert(1)</script>` | XSS 被转义 |

**模块 2（写作 Pipeline）**：
| 2.1.5 | LLM 调用失败 | 断开网络 → 发送消息 | 显示错误提示，不卡死 |
| 2.1.6 | 空消息发送 | 不输入内容直接发送 | 按钮禁用或提示 |
| 2.2.8 | 章节内容为空 | 创建空章节 → 点击阅读 | 显示占位符 |

**模块 3（文风分析）**：
| 3.1.7 | 超长文本分析 | 粘贴 50,000 字 | 截断提示或分段处理 |
| 3.2.5 | 诊断时网络中断 | 点击诊断后断网 | 错误提示，不白屏 |
| 3.4.9 | 改写时 LLM 失败 | 预览改写时服务商 503 | 显示具体错误，保留原文 |

**模块 5（模型配置）**：
| 5.1.8 | 无效 API Key | 输入 "fake-key" → 测试连接 | 显示认证失败 |
| 5.1.9 | 无效 Base URL | 自定义服务输入错误 URL | 显示连接超时/失败 |
