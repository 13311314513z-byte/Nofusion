### 2.2 关键差距总结

**❌ 缺失项（3个）**：
1. **Cover Config 管理页** — 后端 API 完备，前端无独立配置界面，用户只能通过 Chat 指令交互
2. **AI-Tells 独立面板** — 核心 `analyzeAITells` 功能成熟（支持多语言），但前端没有独立入口查看章节的 AI 痕迹指标
3. **Detection Insights 面板** — `analyzeDetectionInsights` 提供结构化诊断洞察，前端未暴露

**⚠️ 间接可用项（2个）**：
1. **ConsolidatorAgent** — 状态合并功能在 Pipeline 中自动触发，但用户无法查看合并前后的差异
2. **AI-Tells** — 在 Audit 流程中间接使用，但无法单章独立检测

## 六、风险与注意事项

4. **AI-Tells 多语言支持**：`analyzeAITells` 已支持 `language` 参数，前端接入时需确保传递正确的章节语言设置
