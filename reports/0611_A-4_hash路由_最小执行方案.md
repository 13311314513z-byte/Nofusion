## 二、唯一改动

### 文件：`packages/studio/src/App.tsx`

**修改前（line 83）：**
```typescript
    toTruth: (bookId: string) => setRoute({ page: "book", bookId, section: "truth" }),
```

**修改后：**
```typescript
    toTruth: (bookId: string) => setRoute({ page: "truth", bookId }),
```

---

## 三、为什么这一行就够了

| 依赖项 | 状态 | 说明 |
|--------|:----:|------|
| `parseHash("#/truth/:bookId")` | ✅ 已就绪 | 返回 `{ page: "truth", bookId }` |
| `routeToHash({ page: "truth", bookId })` | ✅ 已就绪 | 返回 `"#/truth/:bookId"` |
| `HASH_PAGES.has("truth")` | ✅ 已就绪 | 允许写入 URL hash |
| `App.tsx` 渲染分支 | ✅ 已就绪 | `<BookWorkspace section="truth">` |
| `BookWorkspace` 的 `"truth"` section | ✅ 已就绪 | `BookTruthSection` 已注册 |
| `BookSection` 类型 | ✅ 已就绪 | `"truth"` 已在 union 中 |

这一行改动只是让**程序导航**（点击按钮）生成的 URL 与**手动输入**的 URL 保持一致（`#/truth/:bookId`），不再走旧的 `#/book/:bookId/truth` 路径。
