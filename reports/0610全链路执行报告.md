## 五、仍存在的风险与建议

### 🔴 高风险（建议优先处理）

| 风险 | 说明 | 建议 |
|------|------|------|
| API Key 完整返回前端（P1-5） | `GET /services/:service/secret` 返回完整 Key | 改为仅返回 `{ hasApiKey, keyPreview }` |
| 会话端点 404（P1-11） | `GET /books/{id}/sessions` 返回 404 | 排查路由注册或前端调用路径 |
| Chrome 密码管理器接管 API Key | 未使用真实 Chrome 验证 | 人工验收 Key 输入/保存/切换全流程 |

### 🟡 中风险

| 风险 | 说明 |
|------|------|
| 文件锁 TOCTOU（P1-1） | 高并发场景仍有 EEXIST 风险 |
| 新建书籍无进度反馈（P2-9） | 用户长任务期间无法判断是否卡死 |
| RhetoricHighlightEditor 等 3 组件悬空（P2-5/6/7） | 功能不可见 |
| SSE 事件未对齐（P3-6） | 6 种后端事件被前端丢弃 |
| 英文分析硬编码 `language: "zh"`（P3-3） | 文本语言检测缺失 |

### 🟢 低风险

| 风险 | 说明 |
|------|------|
| 无全局请求超时（P3-4） | 部分请求可能无限等待 |
| 15 个悬空后端端点（P3-5） | 无前端入口，不影响主流程 |
