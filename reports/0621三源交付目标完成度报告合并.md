- agent 返回空 `responseText` → fallback 调用 `chatCompletion` 探测 upstream
- 探测抛出异常 → 外层 catch 返回 500 / `AGENT_ERROR`

**期望行为**:
- 探测抛出异常 → 单独 catch → 返回 502 / `AGENT_EMPTY_RESPONSE`

**修复方向**:
```typescript
// 在 agent.ts 的空响应处理段中：
if (!result.responseText) {
  try {
    const probe = await chatCompletion(client, model, [{ role: "user", content: "ping" }], { maxTokens: 16 });
    // ... handle probe success
  } catch (probeError) {
    const message = probeError instanceof Error ? probeError.message : String(probeError);
    return c.json({
      error: { code: "AGENT_EMPTY_RESPONSE", message },
      response: message,
    }, 502);
  }
}
```

**验证**: `vitest run -t "probes the upstream when the agent returns empty text"` → PASS
