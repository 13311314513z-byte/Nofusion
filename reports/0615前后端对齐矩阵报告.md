## 五、核心发现

1. **Agent → 端点覆盖良好**：47 个 Agent 中 35 个（~75%）有对应端点或管线入口
2. **端点 → 前端覆盖不足**：~140 端点中约 25 个（~18%）无前端消费者
3. **0615 新增 Agent 的前端覆盖最弱**：VoiceProfile、EventChain、SceneTemplate 全无前端
4. **管线内 Agent 无需前端**：12 个 Agent 完全自动运行，属于正常设计
5. **样式蒸馏已修复类型但未接入路由**：P1 问题
