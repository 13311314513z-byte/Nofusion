## 七、否决与暂缓建议

| 需求 | 来源 | 否决理由 |
|------|------|----------|
| 全量 API Key 加密存储 | DS | 当前 OS 文件权限 + 项目隔离已足够；全量加密需引入密钥管理 |
| 移动端适配 | DS | 桌面端设计，移动适配工作量 > 5 天且非核心场景 |

| 第三方登录/OAuth | DS | 本地单用户应用，OAuth 改造成本远高于收益 |
| D3.js 图表库引入 | DS | 纯 CSS/SVG 已满足需求；引入 D3 增加 100KB+ bundle |
